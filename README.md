# 3D Model of Bus with WebGL

## Requirements

Language: HTML, JavaScript using WebGL

IDE: Notepad++

Browser: Google Chrome


## Features
1. Indices and Colors
2. MVP (Model, View and Perspective)
3. Mouse event and Key-press event
4. Texture mapping
